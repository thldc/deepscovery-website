기업의 목적

AGI for Dating Agent

Affinia는 인공지능 기술을 활용하여 데이팅 경험을 혁신하는 것을 목표로 합니다. 단순한 데이팅 앱이 아닌, 개인화된 AI 에이전트가 사용자의 취향과 성격을 분석하여 최적의 매칭을 제공합니다.

비전: 개인화된 AI 매칭 기술로 더 의미 있는 관계 형성을 지원합니다.

미션: 첨단 AI 기술을 활용하여 전통적인 데이팅 앱의 한계를 뛰어넘는 혁신적인 서비스를 제공합니다.

왜 지금 필요한가

AI 기술의 발전

LLM과 컴퓨터 비전 기술의 급속한 발전으로 개인화된 매칭 알고리즘 구현이 가능해졌습니다.

사용자 경험의 한계

기존 데이팅 앱의 Swipe 기반 UI는 표면적인 정보에만 의존하여 의미 있는 매칭에 한계가 있습니다.

디지털 관계의 증가

팬데믹 이후 온라인 관계 형성에 대한 수요가 증가하였으며, 더 정교한 매칭 시스템이 필요합니다.

개인화 요구 증가

사용자들은 자신의 취향과 가치관에 맞는 더 개인화된 매칭 서비스를 원하고 있습니다.

  • 온라인 데이팅 시장 규모: 2023년 기준 약 80억 달러

  • 주요 시장: 북미, 유럽, 동아시아

제품 개발 로드맵

Phase 0: 기반 기술 구축

Azure AI 클라우드 상에서 Langraph와 Prompt Engineering 기법을 활용하여 자체 모델 개발 없이 LLM과 고객 데이터베이스를 유기적으로 통합해 데모 버전을 신속하게 구축합니다.

핵심 기술: Langraph, Prompt Engineering, Azure AI Cloud

개발 기간: 2개월

Phase 1: Beauty Score 측정

컴퓨터 비전 기술을 활용하여 사용자의 얼굴 특징을 분석하고 객관적인 미적 점수를 산출합니다. 이 과정에서 사용자 선호도와 문화적 맥락을 고려한 다차원적 분석이 이루어집니다.

핵심 기술: 딥러닝 기반 얼굴 인식, 특징 추출 알고리즘

개발 기간: 3개월

Phase 2: Facial Features Classification

사용자의 얼굴 특징을 세부적으로 분류하고 선호도와 매칭하는 시스템을 개발합니다. 이를 통해 외모 기반의 초기 매칭 정확도를 향상시킵니다.

핵심 기술: 얼굴 구조 분석, 선호도 매핑 알고리즘

개발 기간: 4개월

Phase 3: SNS Image에서 성격 특성 추출

사용자의 SNS 이미지를 분석하여 성격, 취미, 라이프스타일 등의 특성을 추출합니다. 이 데이터는 더 깊은 수준의 매칭에 활용됩니다.

핵심 기술: 이미지 컨텍스트 분석, 성격 추론 알고리즘, LLM 통합

개발 기간: 5개월

중요성: SNS 이미지는 사용자의 실제 생활과 취향을 반영하므로, ColdStart 문제 해결에 효과적입니다.

Phase 4: 관계 지속성 및 만족도 예측

축적된 데이터를 기반으로 잠재적 커플의 관계 지속 기간과 상호 만족도를 예측하는 모델을 개발합니다. 이를 통해 의미 있고 지속 가능한 관계 형성을 촉진합니다.

핵심 기술: 관계 역학 모델링, 예측 알고리즘, 피드백 시스템

개발 기간: 6개월

Phase 5: 자기복제 Agent와 가상 대화

사용자의 대화데이터를 분석해 자기복제 Agent를 생성합니다. 이 Agent들은 서로 자동으로 대화하며 상호만족도를를 평가하고 이로써 사용자의 시간을 절약하고 매칭 정확도를 크게 높일 수 있습니다.

핵심 기술: 대화데이터 기반 에이전트 생성, 멀티 에이전트 상호작용

UI/UX에서 Swipe 다음 패러다임이 LLM인 이유

인간-컴퓨터 상호작용(HCI)의 진화

2000~2010: 그래픽 기반 인터페이스 (마우스/클릭)

2010년대: 터치 인터페이스 (스와이프/핀치)

2020년대: 대화형 인터페이스 (자연어 처리)

ChatGPT의 성공은 사용자 인터페이스의 근본적인 변화를 보여줍니다. 복잡한 메뉴, 버튼, 스와이프 대신 '대화'를 통해 의도를 전달하는 방식이 주목받고 있습니다. 이는 인간의 자연스러운 소통 방식에 더 가깝습니다. 생각, 감정, 필요를 언어로 표현하는 과정에서 더 깊은 수준의 상호작용이 가능해집니다.

GPT와 같은 대규모 언어 모델은 사용자의 복잡한 요구사항을 이해하고 맥락에 맞는 응답을 제공할 수 있습니다. '나와 잘 맞는 사람을 찾아줘'와 같은 요청에 대해 다차원적인 분석이 가능해집니다. "취미가 같은 사람" "성격이 비슷한 사람" 등의 미묘한 뉘앙스까지 포착할 수 있습니다.

Affinia의 접근: 기존 스와이프 UI의 한계를 넘어, LLM 기반의 대화형 에이전트를 통해 사용자의 깊은 수준의 선호도와 필요를 파악하고 최적의 매칭을 제공합니다.

Tinder/Glam은 왜 추천 고도화 서비스라고 볼 수 없는가?

알고리즘의 한계

Agi 수준의 Dating Agent에 도달하려면 아래 데이터를 정답(Label/Reward Signal)으로 학습하는 강화학습 구조가 필수적입니다. 하지만 Tinder/Glam의 UI는 사용자가 앱 이탈 후 실제 결과 데이터와 고품질 피드백을 수집하기 어려워 추천 정밀도 향상이 구조적으로 제한됩니다. 반면 저희 서비스는 대화형 UX로 이러한 데이터를 자연스럽게 수집하여 지속적인 개선이 가능합니다.

  1. 실제 만남 이후, 양 당사자가 서로에 대해 평가한 만족도 점수의 합 (0~100점 기준)
  2. 실제로 관계가 형성되었을 경우, 해당 관계가 유지된 기간

UI/UX의 한계

기존 서비스들은 수동적인 스와이프 인터페이스에 의존하여, 사용자의 복잡한 필요와 선호를 표현하는 데 한계가 있습니다. 이는 사용자 경험의 깊이를 제한하고, 매칭의 질적 향상을 저해합니다.

Affinia의 차별점

Affinia는 단순한 매칭 앱이 아닌, 사용자의 복잡한 선호도와 성격을 이해하는 AGI 기반 에이전트/비서로 포지셔닝합니다. 이를 통해 기존 데이팅 앱의 한계를 근본적으로 극복합니다.
